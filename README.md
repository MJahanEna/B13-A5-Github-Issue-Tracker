1️. What is the difference between var, let, and const?
=> var is function-scoped and can be redeclared, let is block-scoped and can be updated, while const is block-scoped and cannot be reassigned.

2️. What is the spread operator (...)?
=> It allows an array or object to be expanded into individual elements or properties.

3️. What is the difference between map(), filter(), and forEach()?
=> map() creates a new array by transforming every element, filter() creates a new array with only elements that pass a test, and forEach() simply executes a function for each element without returning a new array.

4️. What is an arrow function?
=> A shorter way to write functions using the => syntax that also lexically binds the this value.

5️. What are template literals?
=> Template literals are strings enclosed in backticks (`) that allow for easy multi-line text and embedding variables using ${expression}
